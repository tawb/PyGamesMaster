## 📌 About

This repository contains a series of Python mini-projects developed to explore core programming concepts through interactive games and visual graphics. From classic arcade games to creative art generated with Turtle, each project focuses on different aspects of programming — event handling, collision detection, game loops, object-oriented design, and graphical rendering.

## 🕹 Projects

### 🎮 Games

| Project | Description |

## ⚙ Installation

### Prerequisites
- Python 3.8+
- Pygame (for Pong and Snake)

### Setup

**1. Clone the repository**
```bash
git clone https://github.com/<your-username>/python-games-collection.git
cd python-games-collection
```

**2. Install Pygame**
```bash
pip install pygame
```

**3. Run any project**
```bash
# Run a standalone script
python coffemashin.py
python turtulerace.py
python turtulescetch.py

# For zipped projects, extract first
unzip pong.zip
cd pong
python main.py
```

> *Turtle-based projects use Python's built-in `turtle` module — no extra installation needed.*

## 🧠 Concepts Covered

- **Game Loops** — continuous update-render cycles driving real-time gameplay
- **Event Handling** — keyboard and mouse input for player controls
- **Collision Detection** — detecting interactions between game objects
- **State Management** — tracking scores, lives, levels, and game-over conditions
- **Object-Oriented Programming** — modular, reusable classes for game components
- **Turtle Graphics** — programmatic drawing, generative art, and visual patterns
- **User Input Simulation** — text-based interfaces (coffee machine) and keyboard-driven sketching
